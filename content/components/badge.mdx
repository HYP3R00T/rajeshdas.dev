## Overview

The Badge component is a lightweight, reusable label for highlighting tags, categories, statuses, and other compact information across your application.

## Usage

```astro
---
import Badge from "@/components/ui/Badge.astro";
---

<Badge>New</Badge>
```

## Variants

Three visual styles are available:

<div class="flex gap-2 flex-wrap">
  <Badge variant="default">Default</Badge>
  <Badge variant="outline">Outline</Badge>
  <Badge variant="accent">Accent</Badge>
</div>

```astro
<Badge variant="default">Default</Badge>
<Badge variant="outline">Outline</Badge>
<Badge variant="accent">Accent</Badge>
```

### Default

Bordered badge with secondary text color. Best for general-purpose tagging.

### Outline

Transparent background with border only. Use for less prominent labels.

### Accent

Filled with accent color. Perfect for highlighting important or featured items.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "outline" \| "accent"` | `"default"` | The visual style variant |
| `class` | `string` | `undefined` | Additional CSS classes |

## Usage Examples

### Blog Tags

```astro
<Badge variant="default">astro</Badge>
<Badge variant="default">devops</Badge>
<Badge variant="default">tutorial</Badge>
```

### Status Badge

```astro
<Badge variant="accent">Featured</Badge>
```

### Filter Tags

```astro
{tags.map((tag) => <Badge variant="outline">{tag}</Badge>)}
```

## Accessibility

- The badge is a semantic `<span>` element with no interactive functionality
- Use appropriate context in your markup to convey meaning
- Pair with descriptive text when used for status indicators
