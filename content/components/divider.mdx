## Overview

The Divider component is a simple, semantic separator for visually organizing content. It supports both horizontal and vertical orientations.

## Usage

```astro
---
import Divider from "@/components/ui/Divider.astro";
---

<Divider />
```

## Horizontal

Default orientation for separating vertical content:

```astro
<section>Content above</section>
<Divider />
<section>Content below</section>
```

## Vertical

For separating horizontal content:

```astro
<div class="flex gap-4">
  <div>Left column</div>
  <Divider orientation="vertical" />
  <div>Right column</div>
</div>
```

## Variants

### Default

Subtle border using `border-border`:

```astro
<Divider variant="default" />
```

### Strong

More prominent border using `border-border-strong`:

```astro
<Divider variant="strong" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Direction of the divider |
| `variant` | `"default" \| "strong"` | `"default"` | Visual prominence |
| `class` | `string` | `undefined` | Additional CSS classes |

## Accessibility

- The component uses `aria-orientation` attribute to indicate direction
- Non-interactive, semantic separator element
- Provides visual structure without content
- Complements text-based section breaks like `<hr />`

## Common Patterns

### Section Break

```astro
<div>Previous section content</div>
<Divider />
<div>Next section content</div>
```

### Grid Separator

```astro
<div class="grid grid-cols-2 gap-4">
  <div>Left</div>
  <Divider orientation="vertical" />
  <div>Right</div>
</div>
```

### Form Break

```astro
<div>Form section 1</div>
<Divider />
<div>Form section 2</div>
```
