## Overview

The ButtonGroup component visually connects multiple buttons together, removing borders between adjacent buttons for a unified appearance. It supports both horizontal and vertical orientations.

## Usage

```astro
---
import Button from "@/components/ui/Button.astro";
import ButtonGroup from "@/components/ui/ButtonGroup.astro";
---

<ButtonGroup>
  <Button variant="outline">Left</Button>
  <Button variant="outline">Center</Button>
  <Button variant="outline">Right</Button>
</ButtonGroup>
```

## Horizontal

Default horizontal layout:

<ButtonGroup>
  <Button variant="outline">Archive</Button>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Direction of button arrangement |
| `class` | `string` | `undefined` | Additional CSS classes |

## Accessibility

- Uses `role="group"` to indicate related buttons
- Ensure child buttons have appropriate labels
- Maintains proper keyboard navigation between buttons
