A powerful command-line interface that brings my entire portfolio to your terminal. Built with Python and modern CLI libraries, it provides an interactive way to explore my experience, skills, and projects without leaving your shell.

## Features

- Interactive terminal UI with rich formatting
- Explore skills and expertise
- Browse project showcase
- View recent blog posts
- Navigate with keyboard or mouse
- ANSI color support

## Installation

```bash
uvx rajesh-das
```

## Technologies

- **Python**: Core language
- **Rich**: Beautiful terminal rendering
- **Typer**: CLI framework
- **PyPI**: Package distribution
