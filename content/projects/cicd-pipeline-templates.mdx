A collection of battle-tested GitHub Actions workflows designed for microservices architecture. These templates include security scanning, automated testing, containerization, and multi-environment deployments to accelerate your CI/CD setup.

## Features

- Pre-built workflow templates
- Security scanning integration
- Automated testing pipelines
- Docker image building
- Multi-environment support
- Infrastructure deployment
- Rollback capabilities

## Technologies

- **GitHub Actions**: CI/CD platform
- **Docker**: Containerization
- **Terraform**: Infrastructure as Code
